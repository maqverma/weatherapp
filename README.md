## Setup Details:

## For iOS:

step 1: yarn install
step 2: cd ios && pod install && cd ..
step 3: yarn run ios

## For Android:

step 1: yarn install
step 2: sync gradle project
step 3: yarn run android

Dependencies:
Node >= 18
React-Native >= 0.77.1
React >= 18.3.1
